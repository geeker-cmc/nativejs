### 冒泡排序
>双层遍历，相邻的两个元素相互比较，满足条件的，位置不变，继续跟下一个数字比较，反之，交换位置。
***
##### 数字交换位子的快捷方法
```
let num1 = 12,num2 =13;//交换两个变量的值，利用位运算符
num1 ^= num2;
num2 ^= num1;
num1 ^= num2;
```

>按位运算符将数字转化为二进制，再进行逻辑运算。

##### 按位与(&)
>只有当两个都为1的时候，才返回1 ,类如2&5

```
    0001
 &  0101
 ---------
    0001

//结果为1
```

#### 按位或(|)
>当其中一个为1时，就为1。类如5|3

```
    0101
 |  0011
 ---------
    0111
//结果为7
```
